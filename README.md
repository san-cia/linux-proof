# Installation Notes
> Note: This installation is to use Kali Linux as the main OS and not dual boot!

##### Preparation:
- Install [Rufus](https://rufus.ie/en/), to create a bootable USB.
- Download [Kali Linux](https://www.kali.org/get-kali/#kali-installer-images) ISO image (complete offline installer with customization).
- Run Rufus and select the ISO image.

##### Installation:
- Plug in USB and boot system.
- Select the USB in the Boot Menu and choose graphical installation.
- Configure settings.

##### Post-Installation:
- Once installation is complete, move Kali Linux boot entry to top.
- Login using credentials created during installation.
